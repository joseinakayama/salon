# 🌿 リンパマッサージサロン ランディングページ

リンパマッサージサロンの公式Webサイト（トップページ）です。**Tailwind CSS**を使用したナチュラル＆オーガニックなデザインで、安心感と信頼感を与え、Web予約につなげることを目的としています。

### ターゲット
疲れ、むくみ、冷え、慢性的なだるさなどに悩む30代〜50代の女性

### デザインコンセプト
- **テーマ**: ナチュラル＆オーガニック
- **雰囲気**: 安心感、優しさ、自然体、リラックスできる空間
- **配色**: アースカラー（セージグリーン、サンドベージュ、ソフトなテラコッタ）

### カラーパレット（Tailwind CSS Custom Colors）
```javascript
colors: {
    base: '#F8F6F2',    // 生成り色 / リネン（背景70%）
```

## 🎨 Tailwind CSSの特徴

### ユーティリティファーストアプローチ
- クラスベースのスタイリング
- 高速な開発とプロトタイピング
- レスポンシブデザインが容易

### カスタム設定
```javascript
tailwind.config = {
    theme: {
        extend: {
            colors: {
                base: '#F8F6F2',
                light: '#FDFCFA',
                main: '#8A9A5B',
                sub: '#D4BBA4',
                accent: '#E6A47E',
                text: '#4D4D4D',
            },
            fontFamily: {
                sans: ['Noto Sans JP', 'sans-serif'],
                serif: ['Noto Serif JP', 'serif'],
            },
        }
    }
}
```

### 主な使用クラス
- **レイアウト**: `grid`, `flex`, `container`, `mx-auto`
- **スペーシング**: `p-4`, `m-8`, `gap-6`, `space-y-4`
- **カラー**: `bg-base`, `text-main`, `bg-gradient-to-br`
- **タイポグラフィ**: `font-serif`, `text-4xl`, `font-semibold`
- **エフェクト**: `shadow-lg`, `hover:-translate-y-2`, `transition-all`
- **レスポンシブ**: `md:grid-cols-3`, `lg:text-7xl`

## 🔧 カスタマイズ方法

### 1. 配色変更
`index.html` の `<script>` タグ内のTailwind設定を変更：

```javascript
tailwind.config = {
    theme: {
        extend: {
            colors: {
                base: '#F8F6F2',      // 背景色
                main: '#8A9A5B',      // メインカラー
                sub: '#D4BBA4',       // サブカラー
                accent: '#E6A47E',    // アクセントカラー
                text: '#4D4D4D',      // テキスト色
            }
        }
    }
}
```

### 2. フォント変更
Google Fonts読み込み部分とTailwind設定のfontFamilyを変更。

### 3. コンテンツ編集
`index.html` の各セクション内のテキストを直接編集。

### 4. 画像追加
プレースホルダー部分を `<img>` タグに置き換え：

```html
<!-- 変更前 -->
<div class="w-full h-full bg-gradient-to-br from-main to-sub ...">
    <i class="fas fa-spa text-6xl"></i>
</div>

<!-- 変更後 -->
<img src="path/to/your/image.jpg" alt="説明" class="w-full h-full object-cover">
```

### 5. Tailwindクラスのカスタマイズ
`css/style.css` にカスタムスタイルを追加可能。

## 🎯 Tailwind CSSのメリット

### 1. 高速な開発
- ユーティリティクラスで素早くスタイリング
- カスタムCSSを書く時間を削減
- コンポーネントの再利用が容易

### 2. 一貫性のあるデザイン
- デザインシステムの自動適用
- スペーシング、カラー、タイポグラフィの統一
- レスポンシブデザインが簡単

### 3. メンテナンス性
- HTMLとスタイルが同じ場所に
- クラス名を考える必要がない
- 変更が局所的で影響範囲が明確

### 4. パフォーマンス
- 本番ビルドで未使用クラスを削除（PurgeCSS）
- 最小限のCSSファイルサイズ
- CDN版でも高速

## 💡 Tips & Best Practices

### Tailwind CSSの効率的な使い方
1. **クラスの再利用**: 繰り返し使うスタイルは`@apply`でコンポーネント化
2. **レスポンシブ設計**: モバイルファーストで構築、必要に応じて`md:`や`lg:`を追加
3. **カラーの一貫性**: カスタムカラーを定義して使用
4. **スペーシングの統一**: Tailwindのスペーシングスケール（4px単位）を活用

### パフォーマンス向上のコツ
1. 画像は適切なサイズに圧縮（TinyPNGなど）
2. WebP形式の使用を検討
3. 重要なコンテンツを先に読み込む（Above the Fold）
4. 外部スクリプトは必要最小限に

### SEO対策
1. 適切な`<title>`と`<meta description>`
2. 見出しタグ（h1, h2, h3）の階層構造
3. 画像の`alt`属性を必ず設定
4. 構造化データ（JSON-LD）の追加を検討
